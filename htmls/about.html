About The Project



The Random Quote Generator is a a simple API created by Robinson Chege who is a student at the Moringa school. The API showcases what he has learnt during the 5weeks course about the basic but important principles of Javascript html and css.


The project simply has 4 html elements all which have been placed all in one directory called htmls. It also uses an external styling with an external css directory with a file called styles.css. It also uses javascript to bring user interactivity.
